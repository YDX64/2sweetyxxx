MySQL Database Kurulumu

Database Bilgileri

  • Database Name: dating_db
  • Tables: 24 tablo
  • Schema File: mobile-app/Gommet Database 1.5/Gomeet.sql
  • Size: ~228 KB (1,057 satır)

🎯 Seçenek 1: Coolify MySQL Service

Coolify'da yeni MySQL database oluşturun:

Coolify Dashboard 
1. Resources → + New Resource → Database
2. MySQL 8.0 seçin
3. Yapılandırma:
   Name: 2sweety-mysql
   Database: dating_db
   Username: dating_user
   Password: [güçlü şifre oluştur]
4. Create Database

Bağlantı Bilgilerini Kaydet

Database oluştuktan sonra şu bilgileri not edin:

  • Host (genellikle 2sweety-mysql)
  • Port: 3306
  • Database: dating_db
  • Username: dating_user
  • Password: [oluşturduğunuz şifre]

📥 SQL Dosyasını Import Etme

Database hazır olduktan sonra schema'yı import edin:

Yöntem 1: Bash Script ile
database-setup.sh 
# Environment variables ayarla
export DB_HOST=localhost
export DB_PORT=3306
export DB_NAME=dating_db
export DB_USER=dating_user
export DB_PASSWORD=your_password
export DB_ROOT_PASSWORD=root_password
export SQL_FILE=/path/to/Gomeet.sql

# Script'i çalıştır
cd "Gomeet Admin Panel 1.5"
./database-setup.sh
Yöntem 2: MySQL Command Line
MySQL Import 
# SQL dosyasını import et
mysql -u dating_user -p dating_db < "mobile-app/Gommet Database 1.5/Gomeet.sql"

# Doğrulama
mysql -u dating_user -p dating_db -e "SHOW TABLES;"
Yöntem 3: phpMyAdmin
  1. phpMyAdmin'e giriş yap
  2. dating_db database'ini seç
  3. Import sekmesine git
  4. Gomeet.sql dosyasını seç
  5. Go butonuna tıkla
