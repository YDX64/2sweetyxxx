6 🛠️ Yaygın Sorunlar ve Çözümleri
❌ Sorun 1: "ERR_CERT_AUTHORITY_INVALID"
Çözüm: Servers → Proxy → Advanced → CA SSL Certificate → Regenerate Certificate
❌ Sorun 2: "SSL alınamıyor"
Çözüm: DNS kayıtlarını tekrar kontrol edin, 15 dakika daha bekleyin
❌ Sorun 3: "Domain erişilemiyor"
Çözüm: VPS'de port 80 ve 443'ün açık olduğunu kontrol edin
# Port kontrolü: sudo ufw status sudo ufw allow 80 sudo ufw allow 443 # DNS kontrolü: dig 2sweety.com
