💡 Static Site Redirect Nedir?

Static Site Coolify'da sadece HTML dosyaları sunan bir servis türüdür. Bizim kullanacağımız yöntem:

  • Basit bir HTML dosyası ile JavaScript redirect yapacağız
  • Bu HTML dosyası alternatif domainlerde (2sweety.me, 2sweety.net) çalışacak
  • Sayfa yüklendiğinde otomatik olarak ana domain'e (2sweety.com) yönlendirecek
  • SEO dostu 301 redirect yapacak

Redirect HTML Dosyası Oluşturma

Ana yönlendirme dosyasını oluşturacağız:

📄 Dosya adı: index.html
<!DOCTYPE html> <html lang="tr"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>2sweety - Yönlendiriliyor...</title> <!-- SEO için 301 redirect --> <meta http-equiv="refresh" content="0;url=https://2sweety.com"> <link rel="canonical" href="https://2sweety.com"> <style> body { font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); margin: 0; padding: 0; display: flex; justify-content: center; align-items: center; min-height: 100vh; color: white; } .container { text-align: center; background: rgba(255, 255, 255, 0.1); padding: 40px; border-radius: 20px; backdrop-filter: blur(10px); } .spinner { border: 4px solid rgba(255, 255, 255, 0.3); border-radius: 50%; border-top: 4px solid white; width: 50px; height: 50px; animation: spin 1s linear infinite; margin: 20px auto; } @keyframes spin { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } } </style> </head> <body> <div class="container"> <h1>💕 2sweety</h1> <div class="spinner"></div> <p>Ana siteye yönlendiriliyorsunuz...</p> <p><a href="https://2sweety.com" style="color: white;"> Otomatik yönlendirme çalışmıyorsa buraya tıklayın </a></p> </div> <script> // JavaScript redirect (daha hızlı) window.location.replace('https://2sweety.com'); // Fallback redirect setTimeout(function() { if (window.location.hostname !== '2sweety.com') { window.location.href = 'https://2sweety.com'; } }, 1000); </script> </body> </html>

7

Deploy ve Test

Servisi deploy edip test edelim:

🚀 "Deploy" butonuna basın ⏳ Build process tamamlanmasını bekleyin (1-2 dakika) ✅ "Running" statusunu gözlemleyin 🔍 Logs'lardan hata olmadığını kontrol edin

🎯 Test Etme:

  1. 2sweety.me → otomatik olarak 2sweety.com'a yönlendirmeli
  2. 2sweety.net → otomatik olarak 2sweety.com'a yönlendirmeli
  3. SSL sertifikaları → tüm domainlerde yeşil kilit gözükmeli
  4. Yönlendirme hızı → 1 saniyeden az sürmeli
8

DNS Son Kontrol

Tüm domainlerin DNS ayarlarını doğrulayın:

# Terminal'de test edin: nslookup 2sweety.me nslookup 2sweety.net nslookup 2sweety.org nslookup 2sweety.io # Hepsi 45.9.190.79 IP'sini göstermeli

🔧 DNS Propagation Kontrolü:

DNS değişiklikleri 5-60 dakika arasında yayılır. Test siteleri:

  • whatsmydns.net - Global DNS propagation
  • dnschecker.org - DNS record verification

🚨 Sık Karşılaşılan Sorunlar

  • Build Failed: index.html dosyasında syntax hatası olabilir
  • SSL Error: DNS propagation tamamlanmamış olabilir
  • Redirect Loop: Ana domain'i (2sweety.com) redirect servisine eklemeyin
  • 404 Error: index.html dosyası root directory'de olmalı
